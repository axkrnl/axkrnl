# Contributing

Thank you for contributing to this project.

## Contributor License Agreement

All contributions are subject to the terms of the Contributor
License Agreement described in `CLA.md`.

By submitting a contribution, you confirm that you agree to
those terms.

## Contribution Process

1. Fork the repository
2. Create a feature branch
3. Commit changes with clear messages
4. Submit a pull request

## Code Standards

* Follow the project's coding style
* Include tests where appropriate
* Ensure code compiles and passes checks

## License of Contributions

Unless explicitly stated otherwise, any contribution intentionally
submitted for inclusion in the project shall be licensed under
the project's open-source license, subject to the Contributor
License Agreement.
